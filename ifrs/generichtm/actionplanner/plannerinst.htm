To use the Action Planner you first need to download the file 'action_planner.zip. To do this follow the instructions below:

1. Click OK.
2. Select Open this file from its current location.
3. Click the Extract button in your zip software.
4. Select the location as 'C:\'.
5. Click Extract.
6. Then close the zip software.

If you are working in XP follow these instructions:

1. Click OK.
2. Click Extract all files.
3. Click next.
4. Select the location as 'C:\'.
5. Click Finish.
6. Then close the zip software.

Next time you click on Action Planner, a Word document will open. Once you have made your notes, save these in the default location and using the default word document name, i.e. click 'Save', not 'Save as...'. When you work through a module the Action Planner button will access these notes assuming you are working on the same computer.
