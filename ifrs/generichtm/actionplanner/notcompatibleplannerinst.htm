If you have not already downloaded the action planner, please follow the instructions below to download and open the planner, otherwise close this window and open your downloaded version manually for editing:

1. Click OK.
2. Select Open this file from its current location.
3. Click the Extract button in your zip software.
4. Select the location as 'C:\'.
5. Click Extract.
6. Then close the zip software.
7. Open the planner manually from the location you saved it to.

If you are working in XP or Vista follow these instructions:

1. Click OK.
2. Click Extract all files.
3. Click next.
4. Select the location as 'C:\'.
5. Click Finish.
6. Then close the zip software.
7. Open the planner manually from the location you saved it to.
