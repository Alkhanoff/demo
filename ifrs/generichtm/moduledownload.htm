To work offline you first need to download a zipped up version of the module onto your hard drive. To do this follow the instructions below:

1. Click OK.
2. Select Open this file from its current location.
3. Click the Extract button in your zip software.
4. Select a location on your hard drive where you want the e-learning extracted.
5. Click Extract.
6. Click launch.htm from your install directory to run the course.
